User Routes

Method Path Description
POST /register Register a new user.
POST /login Login a user that's already registered.
GET /users To get all the users that are registered
GET /user/userID Get a user by using the unique userID.
DELETE /user/:userID Delete a house by using it's unique userID.
PUT /user/:userID Update an existing user profile by using the the userID.

House Routes

Method Path Description
GET /houses Get all the houses.
GET /house/:houseID Get a single house by using the unique houseID.
POST /house Add a house to the API.
PUT /house/:houseID Update a house by making use of the unique houseID.
DELETE /house/:houseID Delete a single house by using the houseID.
